分页

要实现分页，必须知道 某一页的 数据 从哪里开始 到哪里结束

需要的变量数据

  1. 数据总数
  2. 页面大小：每页显示的数据量
  3. 总页数
  4. 当前页码
  5. 当前页的对象集合（就是这个页里的数据）

mysql分页：

假设每页显示10条数据

mysql: 从0开始计数

结论：

分页：第n页的数据： 第(n-1)10+1条 – 第n10条

MYSQL实现分页可以使用

第0页: select * from student limit 0,10 ;

第1页: select * from student limit 10,10 ;

第2页: select * from student limit 20,10 ;

第n页: select from student limit n10,10

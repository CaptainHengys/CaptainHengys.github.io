三层优化

三层优化

1. 加入接口

建议面向接口开发：先接口-再实现类(在service、dao加入接口)

接口与实现类的命名规范

接口所在的包： xxx.service xx.dao

实现类所在的包：xxx.service.impl xx.dao.impl

以后使用接口/实现类时，推荐写法：

接口 x = new 实现类();

IStudentDao studentDao = new StudentDaoImpl();

2. DBUtil通用的数据库帮助类

帮助类一般建议写在xx.util包

dao和DBUtil的区别：

  • dao 是处理特定 类的 数据库操作类

  • DBUtil是通用 数据库操作类
